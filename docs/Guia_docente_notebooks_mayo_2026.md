# Guía docente de uso de notebooks — Mayo 2026

## Secuencia recomendada

Cada sesión se imparte en tres fases:

1. **Demostración guiada**: el docente ejecuta y explica el notebook.
2. **Práctica supervisada**: el alumnado modifica parámetros y replica resultados.
3. **Ejercicio final**: se completa la sección de ejercicios y conclusiones.

## Evaluación sugerida por notebook

- Ejecución correcta del código: 30 %
- Interpretación de métricas/gráficos: 30 %
- Conclusiones de negocio: 25 %
- Limpieza y documentación del notebook: 15 %

## Entregables

Cada estudiante debe entregar el notebook ejecutado con:
- resultados visibles,
- gráficos generados,
- comentarios markdown propios,
- conclusiones finales.
